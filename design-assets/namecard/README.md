# Linus business-card QR assets

Portable QR assets extracted from the Hailan website repository for reuse in another repository.

## Files

| File | Use | Format |
|---|---|---|
| `linus-business-card-qr-print.svg` | Preferred source for professional printing | Vector SVG, black |
| `linus-business-card-qr-print-2000px.png` | High-resolution print fallback | 2000 × 2000 transparent PNG |
| `linus-profile-qr-web.svg` | Website/UI version linking to the Linus contact page | Vector SVG, Hailan ink color |
| `linus-wecom-qr-500px.png` | WeCom contact QR | 500 × 500 transparent PNG |

## Destination

The profile/business-card QR is intended to open:

`https://hailanworld.com/linus`

Regenerate the profile QR if the public URL changes. Test every printed proof with multiple phones before mass printing. Preserve the QR quiet zone, do not crop the image, and avoid placing it on a low-contrast background.

The WeCom image is personal contact material. Keep it only in repositories and channels where that contact information is intended to be shared.
